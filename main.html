Course Requirements

  1. Prerequisites
    - No prior programming experience required
  2. Software Needed
    - Visual Studio Code (free text editor)

Learning Outcomes

By the end of this course, you will be able to:

  1. Understand how the internet and web technologies work
  2. Create structured HTML documents with proper semantics
  3. Build multipage websites with navigation
  4. Organize files and folders for web projects
  5. Create interactive forms for user input
  6. Add multimedia content to your websites
  7. Deploy your website live on the internet
