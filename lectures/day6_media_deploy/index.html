1.) HTML Media: First Principle

The web is more than text and images. Historically, video and audio required plugins like Flash or QuickTime, which were inefficient and insecure.

The Core Problem: How to embed video and audio in a standardized, native, plugin-free way?

Solution: HTML5 <video> and <audio> tags.

2.) Embedding Video

Basic video embedding:

<video src="flower-video.mp4" controls></video>

Result: a functional video player with play/pause, volume, timeline, and fullscreen controls.

Advanced: Multiple Sources & Poster

To ensure browser compatibility, provide multiple formats and a poster image:

<video width="320" height="240" controls poster="images/video-thumbnail.jpg">
  <source src="video.webm" type="video/webm">
  <source src="video.mp4" type="video/mp4">
  Sorry, your browser does not support embedded videos.
</video>

Accessibility with Captions

Use <track> for subtitles or captions:

<video controls>
  <source src="video.mp4" type="video/mp4">
  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
  <track src="subtitles_es.vtt" kind="subtitles" srclang="es" label="Espa√±ol">
</video>

3.) Embedding Audio

Embed sound files with player controls:

<audio src="audio/theme-song.mp3" controls></audio>

Supports autoplay, loop, and muted attributes, similar to video.

4.) Deployment & Hosting

Deploy your website using GitHub Pages, Netlify, or other hosting services. This makes your work publicly accessible.

Example: GitHub Pages automatically hosts your index.html from a repository.

5.) Copyright & HTML Entities

Two ways to create the ¬© symbol:

1. Using Entity Name (Recommended)

&copy;

Example in footer:

<footer>
  <p>Copyright &copy; 2025 My Awesome Website. All Rights Reserved.</p>
</footer>

2. Using Entity Number

&#169;

Example in footer:

<footer>
  <p>Copyright &#169; 2025 My Awesome Website. All Rights Reserved.</p>
</footer>

Output of both the HTML Entities will be:- Copyright ¬© 2025 My Awesome Website. All Rights Reserved.

6.)üìñ Glossary of Terms

  1. <video>: Embeds a video file directly into the webpage with native controls.
  2. <audio>: Embeds an audio file with play, pause, and volume options.
  3. <source>: Specifies multiple media files (different formats) for <video> or <audio>.
  4. controls: Attribute that displays play, pause, timeline, and volume buttons for media.
  5. autoplay: Automatically starts playing the media when the page loads (often requires muted).
  6. loop: Restarts the media once it finishes, repeating endlessly.
  7. muted: Starts the media without sound (can be unmuted by the user).
  8. poster: Defines an image to show before the video starts playing.
  9. <track>: Adds subtitles or captions for video content, improving accessibility.
  10. &copy; / &#169;: HTML entity to display the ¬© copyright symbol.
  11. Deployment: The process of publishing your website on the internet (e.g., GitHub Pages, Netlify).
  12. Hosting: A service that stores your website files and makes them accessible online.
