1.) What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: Ability to create links that connect web pages, making the web a connected "web" of information.
  • Markup Language: Uses "tags" to structure content. You "mark up" plain text with meaning and structure.

HTML elements form the building blocks of web pages. Each element has a start tag, content, and an end tag (except self-closing tags like <img> and <br>).

Example of a simple HTML element:

<p>This is a paragraph.</p>

3.) Headings: <h1> to <h6>

Headings define titles and subtitles, creating a logical hierarchy on your webpage:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6
  • <h1>: Most important heading, usually used once per page.
  • <h2>: Major section heading.
  • <h3>: Sub-section under <h2>.
  • <h4> to <h6>: Used for sub-subsections and less important headings.

4.) Paragraphs: <p>

Paragraphs group sentences and blocks of text together:

This is the first paragraph. It contains several sentences about a topic.

This is a second, separate paragraph. Browsers display space between paragraphs automatically.

Example:

<p>HTML makes it easy to structure your content.</p>

5.) Horizontal Rules & Line Breaks

The <hr> tag defines a thematic break:

This is the first topic.

This is a completely different topic.

The <br> tag creates a line break:

My Name is Nandan Aghera
I'm from Noida

6.) Lists: <ul>, <ol>, <li>

Lists are used to group related items:

Unordered List:

  • Tea Bag
  • Water
  • Sugar
  • Milk

Ordered List:

  1. First, boil the water in a kettle.
  2. Add tea bag, sugar and milk into it.
  3. Keep boiling it for 5 minutes.
  4. Serve the tea by pouring it into a cup.
<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
</ul>

7.) Anchor Tag & Hyperlinks

The <a> tag creates clickable links:

Visit Coder Army

Target Attribute:

  • _self: Opens link in same window/tab (default)
  • _blank: Opens link in a new window/tab

Example of internal link:

<a href="about.html">About Us</a>
<a href="https://www.coderarmy.in/" title="Go to Coder Army website">Visit Coder Army</a>

8.) Images: <img>

The <img> tag embeds images:

Attributes:

  • src: Path to image (mandatory)
  • alt: Alternative text for accessibility
  • width & height: Specify size of the image

9.) üìñ Glossary of Terms

  1. HTML: HyperText Markup Language, used to structure web content.
  2. HyperText: Links that connect web pages together.
  3. Markup Language: Uses tags to give meaning and structure to text.
  4. Tag: An element that defines content and structure in HTML.
  5. Attribute: Provides additional information about an HTML element (like src or href).
  6. Paragraph: A block of text enclosed in <p> tags.
  7. Heading: Titles/subtitles in HTML (<h1> to <h6>).
  8. List: A collection of items, ordered (<ol>) or unordered (<ul>).
  9. Anchor: The <a> tag used to create hyperlinks.
  10. Image: The <img> tag used to display pictures.
  11. Horizontal Rule: The <hr> tag used to separate content.
  12. Line Break: The <br> tag used to start a new line.
