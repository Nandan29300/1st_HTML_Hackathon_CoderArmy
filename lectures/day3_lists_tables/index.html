1.) Nested Lists

A nested list means placing a list inside another list item. This creates a hierarchy of content.

Example (Unordered List)

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Leafy Greens</li>
      <li>Root Vegetables</li>
    </ul>
  </li>
</ul>

Output:

  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Leafy Greens
    • Root Vegetables

Example (Ordered List with Sub-steps)

<ol>
  <li>Step 1
    <ol>
      <li>Substep A</li>
      <li>Substep B</li>
    </ol>
  </li>
  <li>Step 2</li>
</ol>

Output:

  1. Step 1
    1. Substep A
    2. Substep B
  2. Step 2

2.) Types of Lists

  • <ul>: Unordered list
  • <ol>: Ordered list

Example: Unordered List (<ul>)

<ul>
  <li>Tea</li>
  <li>Coffee</li>
  <li>Water</li>
</ul>

Output:

  • Tea
  • Coffee
  • Water

Example: Ordered List (<ol>)

<ol>
  <li>Wake up</li>
  <li>Brush Teeth</li>
  <li>Have Breakfast</li>
</ol>

Output:

  1. Wake up
  2. Brush Teeth
  3. Have Breakfast

3.) HTML Tables

A <table> is used to display data in rows and columns.

Basic Example

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Nandan</td>
    <td>21</td>
  </tr>
  <tr>
    <td>Rohit</td>
    <td>27</td>
  </tr>
</table>

Output:

Name Age
Nandan 21
Rohit 27

4.) Nested Tables

You can place a table inside a <td> cell of another table.

Example

<table border="1">
  <tr>
    <th>Group</th>
    <th>Members</th>
  </tr>
  <tr>
    <td>Group A</td>
    <td>
      <table border="1">
        <tr><th>Name</th><th>Role</th></tr>
        <tr><td>Rohit</td><td>Leader</td></tr>
        <tr><td>Nandan</td><td>Member</td></tr>
      </table>
    </td>
  </tr>
</table>

Output:

Group Members
Group A
Name Role
Rohit Leader
Nandan Member

5.) Table Sections

Tables can be divided into sections for clarity:

  • <thead>: Table header rows
  • <tbody>: Main body rows
  • <tfoot>: Footer rows
  • <caption>: Title/description for the table

Example

<table border="1">
  <caption>Student Marks</caption>
  <thead>
    <tr><th>Name</th><th>Marks</th></tr>
  </thead>
  <tbody>
    <tr><td>Rohit</td><td>90</td></tr>
    <tr><td>Nandan</td><td>85</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>175</td></tr>
  </tfoot>
</table>

Output:

Student Marks
NameMarks
Rohit90
Nandan85
Total175

6.)ğŸ“– Glossary of Terms

  1. Nested List: A list placed inside another list item.
  2. <ul>: Unordered/bulleted list.
  3. <ol>: Ordered/numbered list.
  4. <dl>: Definition list, used with <dt> and <dd>.
  5. <table>: Element to display tabular data.
  6. <thead>: Table header section.
  7. <tbody>: Table body section.
  8. <tfoot>: Table footer section.
  9. <tr>: Table row.
  10. <th>: Table header cell.
  11. <td>: Table data cell.
  12. <caption>: Title or description of a table.
